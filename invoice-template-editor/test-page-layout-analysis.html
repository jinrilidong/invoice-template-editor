列间距设置

列间距规格

  • 列与列间距：16px（每列左右各8px padding）
  • 表格边缘：第一列左边和最后一列右边无额外间距
  • 数据表格：4列，每列25%宽度
  • 信息/项目表格：5列，每列20%宽度
分页支持
分页逻辑： 1. 页面高度限制：752px 2. Section高度计算：动态计算每个section的高度 3. 表格分页：支持表格跨页拆分 4. 分页后页面： - 第一部分：标题 + 表头 + 部分数据行 - 第二部分：剩余数据行 + 小计 + 底部边框 5. Footer：每页都有footer显示
CSS特性支持

支持的CSS特性

  • 换行支持：word-wrap, overflow-wrap, white-space
  • 文本对齐：vertical-align: top
  • 表格布局：table-layout: fixed, border-collapse: collapse
  • 分页控制：page-break-before: always
  • 字体设置：Inter字体族，多种字体大小
  • 颜色系统：#6b7280, #919191, #000000, #d2d2d2
布局特点总结
当前HTML导出页面结构特点： 1. 固定尺寸布局： - 页面尺寸：612×792px (U.S. Letter) - 内容区域：572×752px - 所有元素使用固定像素值 2. 表格驱动布局： - 使用HTML table元素进行布局 - 支持OpenHTMLtoPDF兼容性 - 列宽使用百分比分配 3. 分页支持： - 支持多页内容 - 表格可跨页拆分 - 每页都有完整的footer 4. 响应式文本： - 支持文本换行 - 顶部对齐显示 - 16px列间距 5. 样式一致性： - 与PDF预览完全一致 - 统一的字体和颜色系统 - 精确的间距控制
