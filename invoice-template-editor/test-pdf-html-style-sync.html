PDF与HTML样式同步测试

同步目标
确保PDF模板的样式变动能够同步到HTML导出中，包括：
  • 字体设置（字体族、大小、权重）
  • 行高和换行样式
  • 颜色值
  • 间距和布局
  • 边框和分割线
样式对比分析

PDF预览样式

字体：Inter

行高：1.2

颜色：#6b7280, #919191

间距：2px, 4px gaps

HTML导出样式（修复前）

字体：Arial

行高：未设置

颜色：#6b7280, #919191

间距：2px, 4px gaps

HTML导出样式（修复后）

字体：Inter ✅

行高：1.2 ✅

颜色：#6b7280, #919191 ✅

间距：2px, 4px gaps ✅

修复前后对比
修复前（样式不一致）
字体设置：
body { font-family: Arial, sans-serif; font-size: 12px; /* 缺少 line-height: 1.2 */ }
问题：
  • 使用Arial字体而非Inter
  • 缺少整体行高设置
  • 与PDF预览样式不一致
修复后（样式同步）
字体设置：
body { font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif; font-size: 12px; line-height: 1.2; }
优势：
  • 使用Inter字体与PDF预览一致
  • 添加了行高设置
  • 完全匹配PDF预览样式
技术修复详情

1. 字体族同步

// 修复前 font-family: Arial, sans-serif; // 修复后 font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

2. 行高同步

// 修复前 body { font-size: 12px; /* 缺少行高设置 */ } // 修复后 body { font-size: 12px; line-height: 1.2; /* 与PDF预览一致 */ }

3. 颜色值验证

// 已验证一致的颜色值 color: #6b7280; /* Section titles */ color: #919191; /* Secondary text */ color: #000000; /* Primary text */ color: #d2d2d2; /* Borders */
测试步骤
步骤 1：在PDF预览中调整样式（如字体、行高、颜色等）
步骤 2：导出HTML并检查样式是否同步
步骤 3：对比PDF预览和HTML导出的视觉效果
步骤 4：验证字体、行高、颜色、间距的一致性
步骤 5：测试不同浏览器的渲染效果
预期结果
✅ 修复后应该：
  • HTML导出使用与PDF预览相同的Inter字体
  • HTML导出具有与PDF预览相同的行高设置（1.2）
  • 所有颜色值完全一致
  • 间距和布局完全匹配
  • PDF模板样式变动能自动同步到HTML导出
验证方法
1. 代码审查：检查HTML导出中的字体和行高设置
2. 视觉对比：并排对比PDF预览和HTML导出效果
3. 浏览器测试：在不同浏览器中打开导出的HTML
4. 样式检查：使用浏览器开发者工具检查CSS样式
