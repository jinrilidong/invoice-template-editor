📐 红色区域高度

当前设置

  • 红色区域高度: 743px
  • 红色区域宽度: 572px
  • 红色区域margin: margin: 20px 20px 0px 20px
  • box-sizing: content-box
🔍 溢出原因分析

问题根源

红色区域溢出的原因是margin占用额外空间：

空间占用计算：

  • 红色区域高度: 743px
  • 上边距: 20px
  • 实际占用空间: 743px + 20px = 763px
  • Footer高度: 9px
  • Footer位置: 从763px开始
  • 下边距: 20px
  • 总占用空间: 763px + 9px + 20px = 792px

问题分析：

  • 虽然总高度是792px，符合页面高度
  • 但是红色区域的实际占用空间是763px
  • 这导致红色区域超出了预期的743px高度
  • Footer被推到了页面底部，没有在红色区域下方

💡 解决方案

修复方案

方案1: 调整红色区域高度

  • 将红色区域高度从743px调整为723px
  • 这样实际占用空间: 723px + 20px = 743px
  • Footer从743px开始，高度9px
  • 下边距从752px开始，高度20px
  • 总占用: 743px + 9px + 20px = 772px

方案2: 调整margin设置

  • 将margin改为: margin: 20px 20px 0px 20px
  • 使用box-sizing: border-box
  • 确保高度计算包含margin

方案3: 重新设计布局

  • 将Footer放在红色区域内部
  • 红色区域高度: 792px - 40px = 752px
  • Footer在红色区域底部
